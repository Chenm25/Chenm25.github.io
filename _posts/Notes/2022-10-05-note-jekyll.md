# Jekyll Chripy 主题的使用

## YAML Front Matter 模板

1. 常用模板

   ```yaml
   layout: post
   title: ""
   date: 
   tags: []
   categories: []
   pin: false
   ```

2. 进阶模板示例

   ```yaml
   layout: post
   title: "Jekyll Tutorial"
   date: 2022-10-01 14:00:00 +0800
   tags: [blog, jekyll, tutorial]
   categories: [Blogs, Tutorials]
   pin: false
   math: false
   mermaid: false
   render_with_liquid: false
   ```
